## Getting Started
The following script should install all dependencies and programs needed for the dotfiles.
```
git clone https://github.com/SushiWaUmai/dotfiles.git --recursive
cd dotfiles
chmod +x ./install.sh
./install.sh
```

## Prerequisites
- xorg 
- imlib2
